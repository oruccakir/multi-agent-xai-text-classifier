class TextClassificationPipeline:
    def __init__(self):
        self.intent_agent = IntentClassifierAgent()     # Ajan 1
        self.classification_agent = ClassificationAgent() # Ajan 2
        self.xai_agent = XAIAgent()                      # Ajan 3

    def process(self, text, explain=True):
        intent = self.intent_agent.process(text)           # Dil + Domain
        result = self.classification_agent.process(intent)  # Tahmin
        explanation = self.xai_agent.process(result)         # Açıklama
        return {intent, result, explanation}

TF-IDF Vektörizasyon

  • 30,000 boyutlu sparse vektör
  • Unigram + Bigram (1,2)
  • Sublinear TF + L2 normalizasyon
  • min_df=3, max_df=0.95

Türkçe Morfoloji

  • Agglutinative yapı: gel, geldi, gelmedi, gelmeyecekti
  • Bigram ile dolaylı morfolojik kapsam
t-SNE: 2,000 test örneği, PCA-50 → 2D

SVM Neden En İyi?

  • Maximum-margin yaklaşımı sparse TF-IDF'e uygun
  • 30K boyutlu uzayda lineer ayrılabilirlik iyi
  • L2 normalizasyon margin'ı stabilize ediyor
  • Feature bağımsızlığı varsaymıyor (NB'den farklı)

Negatif Sınıf Problemi

  • Negatif: sadece %11.6 eğitim verisi
  • Transformer recall: %81.5
  • SVM recall: %65.3
  • Decision Tree recall: %37.4

Öğrenilenler & Gelecek

Temel Bulgular

  • Sparse TF-IDF uzayında SVM en iyi klasik model
  • BERTurk tüm veri setlerinde birinci
  • Sınıf dengesizliği → Macro-F1 kritik
  • K-Means ARI = −0.017 → kümeleme yetersiz, supervised gerekli
  • 3 ajanlı modüler mimari farklı görevlere esneklik sağlıyor

Limitasyonlar

  • SMOTE / class-weighted loss uygulanmadı
  • Hiperparametre optimizasyonu (grid search) yapılmadı
  • XAI sadakat metrikleri değerlendirilmedi

Gelecek Çalışmalar

  • SMOTE veya class-weighted LinearSVC ile negatif recall iyileştirme
  • SVM C parametresi ve TF-IDF vocabulary grid search
  • XAI sadakat değerlendirmesi (sufficiency, comprehensiveness)
  • Daha fazla Türkçe veri seti ile genelleme testi
