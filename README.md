# Multi-Agent Explainable AI Text Classification System

## Research Question

"Can a multi-agent architecture effectively route and classify multilingual text, and how do traditional machine learning classifiers compare
to transformer-based deep learning models in terms of accuracy, F1-score, and interpretability when explained through SHAP and LIME?"

## IEEE Conference Paper Title

Multi-Agent Explainable AI Text Classification: A Comparative Study of Classical Machine Learning and Transformer-Based Models with SHAP and LIME
